## 📝 注意事项

### 为什么保留技术标识符？

1. **路由名称**：修改路由会导致现有代码中的导航失效
2. **API 方法名**：修改会导致与后端接口不匹配
3. **文件名**：修改需要更新所有 import 语句
4. **类名**：修改需要更新所有引用

这些技术标识符在代码内部使用，不会直接展示给用户，因此保持不变不影响用户体验。
